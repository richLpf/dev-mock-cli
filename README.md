# u-admin-cli

实现功能：

- 下载基于react、antd、axios的后台管理系统模型

## 快速开始

```
npm i -g u-admin-cli

app-cli create <myapp>

```
将自动在对应目录下新建项目myapp

## 版本更新

## 发布版本

> 登录npm

```
npm login
```

1、修改版本号,执行下面命令

```
npm run publish:patch
```
2、撤回24小时内发布的版本，撤回后24小时内不允许发布

```
npm unpublish u-admin-cli@1.0.2
```
